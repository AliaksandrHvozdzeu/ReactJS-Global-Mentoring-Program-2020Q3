[Live movies.mycloud.by](http://movies.mycloud.by/)

_Definition_

The idea is to create a single page application, which will allow users to search, add and delete movies from the Movies DB database.

## Get started ##

### Install all dependencies

```bash
npm install
```

### Start

```bash
npm start
```

### Run unit tests

```bash
npm run test
```

### Lint your code

```bash
npm run lint
```
